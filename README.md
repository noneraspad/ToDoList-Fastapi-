#### Содержание папок и файлов следующее: 
- extra-hooks (Был необходим для того, чтобы я мог преобразовать проект в обычный exe файл для личного удобства)
- stat (Папка для статичных файлов (fastapi.staticfiles)
- templates (Хранит в себе index.html)
- venv (Виртуальное окружение
- database.py (Необходим для SqlAlchemy)
- db.sqlite (Сама БД)
- main.py (Основа проекта)
- models.py (Необхоидмо для дальнейшей работы с БД)
- requirements.txt (Список необходимых библиотек)
